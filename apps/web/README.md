# @graphrag/web

Next.js 15 + React 19 frontend for graphrag-copilot.

## Run

```bash
pnpm install
pnpm dev
# or from repo root:
make web
```

Then visit http://localhost:3000.

## Lint + typecheck

```bash
pnpm lint        # biome
pnpm typecheck   # tsc --noEmit
pnpm build       # full Next.js build
```
