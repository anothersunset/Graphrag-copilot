# 风险评审纪要

## 1. 评审范围
本次评审覆盖 GraphRAG Copilot 的检索层、知识图谱组件、Multi-Agent 编排以及前端演示链路。

## 2. 风险清单(按优先级)

### P0 - 数据泄露风险
- 风险:企业知识被越权访问
- 缓解:知识分级 + 权限校验 + 审计日志
- 负责团队:安全组

### P1 - 知识图谱组件稳定性
- 风险:Neo4j 故障导致多跳查询不可用
- 缓解:KG 调用增加降级链,失败时跳过 KG 步骤
- 负责团队:架构组

### P2 - LLM 调用超时
- 风险:外部 LLM 抖动影响回答
- 缓解:Multi-LLM 路由 + 规则兜底
- 负责团队:平台组

### P3 - 评估指标退化
- 风险:版本迭代后 Recall@5 下降
- 缓解:每次发布跑 eval/run_eval.py
- 负责团队:算法组

## 3. 结论
评审通过,要求在上线前完成 P0/P1 缓解措施,并将所有降级路径写入测试用例。
